# Bible Journaling Together

말씀을 읽고, 기록하고, 함께 나누는 성경 저널링 커뮤니티 홈페이지입니다.

## 구성

- `index.html` — 홈페이지 마크업
- `styles.css` — 반응형 디자인 스타일
- `script.js` — 모바일 메뉴와 저널링 질문 인터랙션

## 로컬에서 보기

브라우저에서 `index.html` 파일을 열면 바로 확인할 수 있습니다.

## GitHub Pages 배포

1. GitHub 저장소의 **Settings**로 이동합니다.
2. 왼쪽 메뉴에서 **Pages**를 선택합니다.
3. **Build and deployment**에서 Source를 **Deploy from a branch**로 설정합니다.
4. Branch는 `main`, 폴더는 `/root`를 선택합니다.
5. 저장하면 잠시 뒤 홈페이지 URL이 생성됩니다.

## 수정 포인트

- 문의 이메일: `index.html`의 `mailto:hello@example.com` 부분을 실제 이메일로 변경하세요.
- 문구/소개: `index.html`에서 섹션별 텍스트를 원하는 표현으로 바꿀 수 있습니다.
- 색상: `styles.css` 상단의 CSS 변수에서 메인 색상을 조정할 수 있습니다.
