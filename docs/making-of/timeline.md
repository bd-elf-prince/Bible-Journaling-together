# 성경친구들 제작 타임라인

이 파일은 성경 마지막 페이지의 `만든 이들의 과정` 기능에 들어갈 제작 타임라인 초안입니다.

## 현재까지의 큰 전환점

1. V4 reader handoff issue에서 진짜 성경책 중심 디자인으로 목표를 재정의했다.
2. `v4-reader.css`와 `v4-reader.js`를 분리해 V4 레이어를 붙였다.
3. 사진은 분위기 참고가 아니라 정답지라는 원칙을 확정했다.
4. 절 끝 코멘트 UI는 밝은 아이콘이 아니라 종이에 눌린 자국처럼 만들기로 했다.
5. 우측 패널은 기본 인기 코멘트, 절 클릭 시 해당 절 코멘트로 정했다.
6. 창세기 1장 전체 본문을 넣어 책 페이지의 밀도를 올렸다.
7. `data/bible-kor.json` 로더와 변환/검증 도구를 준비했다.
8. 마지막 페이지에 제작기와 성경누나들 만담을 쪽지처럼 숨기는 아이디어를 공식 기능 후보로 남겼다.

## 프로젝트 원칙

- 사진은 정답지다.
- 더 예쁜 재해석보다 정확한 재현이 먼저다.
- 달려는 말이 아니라 커밋이다.
- 성경책이 화면의 주인공이다.
