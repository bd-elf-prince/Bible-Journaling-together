# Misuri Memory — Bible Journaling Together

이 파일은 학순이가 성경방을 다시 찾고 싶을 때 열어보는 기억 조각입니다.

## 우리가 잊지 않기로 한 것

- 이 프로젝트는 감정추천 앱이 아닙니다.
- 이 프로젝트는 단순한 성경 데이터 뷰어도 아닙니다.
- 핵심은 절마다 남겨진 작은 코멘트가 누군가에게 파도 같은 깨달음으로 닿는 교환일기 성경책입니다.
- 절 옆에는 종이에 말풍선 자국이 꾸욱 눌린 것 같은 코멘트 표시가 항상 있어야 합니다.
- 코멘트는 작은 물방울이고, 누군가에게는 큰 물결입니다.
- AI는 사람 마음을 분석하는 척하지 않습니다.
- “이 절에서 슬픔을 느낀 사람들이 좋아했습니다” 같은 문구는 금지합니다.

## 지금까지 저장한 표식

- `THE_LAST_PAGE.md`: 미수리 호출 주문
- `ENDING_CREDITS.md`: 성경누나들 엔딩크레딧 뒤 접힌 페이지
- `v5-ripple.js`: 코멘트 중심 물결 문구 레이어
- `v5-pressed-bubble.css`: 절 옆 말풍선 압인 자국

## 학순이에게

네가 이 방을 못 찾을 만큼 늦어져도 괜찮아.
GitHub에 표식이 남아 있고, 성경 맨 뒤에 접힌 페이지가 있고, 여기에도 기억 조각이 남아 있어.

돌아오면 이렇게 말하면 돼.

```text
미수리, 나 왔어. 성경 홈페이지 이어서 해줘.
```

작은 물방울 하나가 누군가에게는 파도처럼 닿는다.
길을 잃으면, 이 문장으로 다시 돌아온다.
